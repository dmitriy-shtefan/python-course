## Вправа 1. Лічильник від 1 до 5

Інструкція: програма має вивести числа від 1 до 5, кожне з нового рядка, використовуючи цикл `while`.

Приклад вводу/виводу:

```text
Ввід: немає

Вивід:
1
2
3
4
5
```

Рішення:

```python
number = 1

while number <= 5:
    print(number)
    number = number + 1
```

## Вправа 13. Обмеження кількості спроб

Інструкція: користувач має максимум 3 спроби для введення пароля `python123`. Якщо пароль правильний, програма виводить `Вхід виконано`. Якщо 3 спроби використано, програма виводить `Доступ заблоковано`.

Приклад вводу/виводу:

```text
Ввід:
111
222
333

Вивід:
Неправильний пароль
Неправильний пароль
Неправильний пароль
Доступ заблоковано
```

Рішення:

```python
attempts = 0
is_logged_in = False

while attempts < 3 and not is_logged_in:
    password = input("Пароль: ")
    attempts = attempts + 1

    if password == "python123":
        is_logged_in = True
    else:
        print("Неправильний пароль")

if is_logged_in:
    print("Вхід виконано")
else:
    print("Доступ заблоковано")
```

## Вправа 19. Вгадування числа

Інструкція: у програмі задано секретне число `7`. Користувач вводить числа, поки не вгадає. Якщо число менше секретного, програма виводить `Більше`. Якщо більше секретного - `Менше`.

Приклад вводу/виводу:

```text
Ввід:
3

Більше
Менше
Вгадано
```

Рішення:

```python

        print("Більше")
    else:
        print("Менше")

    guess = int(input("Ваш варіант: "))

print("Вгадано")
```

## Вправа 20. Банкомат

Інструкція: програма має працювати до команди `0`. На початку баланс дорівнює `1000`. Команда `1` показує баланс, команда `2` поповнює баланс, команда `3` знімає гроші, якщо сума не більша за баланс.

Приклад вводу/виводу:

```text
Ввід:
1
2
500
3
300
1
0

Вивід:
Баланс: 1000.0
Баланс поповнено
Гроші знято
Баланс: 1200.0
Роботу завершено
```

Рішення:

```python
balance = 1000.0
command = ""

while command != "0":
    print("1 - показати баланс")
    print("2 - поповнити")
    print("3 - зняти")
    print("0 - вихід")

    command = input("Ваш вибір: ").strip()

    if command == "1":
        print("Баланс:", balance)
    elif command == "2":
        amount = float(input("Сума: "))

        if amount > 0:
            balance = balance + amount
            print("Баланс поповнено")
        else:
            print("Некоректна сума")
    elif command == "3":
        amount = float(input("Сума: "))

        if amount > 0 and amount <= balance:
            balance = balance - amount
            print("Гроші знято")
        else:
            print("Операцію відхилено")
    elif command == "0":
        print("Роботу завершено")
    else:
        print("Невідома команда")
```

## Вправа 21. Меню журналу оцінок

Інструкція: програма має працювати до команди `0`. Команда `1` додає оцінку від 1 до 12, команда `2` показує кількість оцінок, команда `3` показує середню оцінку. Якщо оцінок ще немає, середнє рахувати не можна.

Приклад вводу/виводу:

```text
Ввід:
1
10
1
12
2
3
0

Вивід:
Оцінку додано
Оцінку додано
Кількість оцінок: 2
Середня оцінка: 11.0
Журнал закрито
```

Рішення:

```python
total = 0
count = 0
command = ""

while command != "0":
    print("1 - додати оцінку")
    print("2 - показати кількість")
    print("3 - показати середню")
    print("0 - вихід")

    command = input("Ваш вибір: ").strip()

    if command == "1":
        grade = int(input("Оцінка: "))

        if grade >= 1 and grade <= 12:
            total = total + grade
            count = count + 1
            print("Оцінку додано")
        else:
            print("Некоректна оцінка")
    elif command == "2":
        print("Кількість оцінок:", count)
    elif command == "3":
        if count > 0:
            average = total / count
            print("Середня оцінка:", average)
        else:
            print("Оцінок ще немає")
    elif command == "0":
        print("Журнал закрито")
    else:
        print("Невідома команда")
```
