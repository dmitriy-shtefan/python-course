## Вправа 1. Два критерії доступу

Інструкція: користувач вводить вік і відповідь на питання, чи має він квиток. Доступ дозволено, якщо користувачу 18 або більше і він ввів `так`.

Приклад вводу/виводу:

```text
Ввід:
19
так

Вивід:
Доступ дозволено: True
Можна входити
```

Рішення:

```python
age = int(input("Вік: "))
ticket = input("Є квиток? ").strip().lower()

can_enter = age >= 18 and ticket == "так"

print("Доступ дозволено:", can_enter)

    print("Можна входити")
else:
    print("Вхід заборонено")
```

## Вправа 13. Перевірка промокоду

Інструкція: користувач вводить суму покупки і промокод. Знижка 15% застосовується, якщо сума від 500 і промокод `python15` або якщо сума від 2000 незалежно від промокоду.

Приклад вводу/виводу:

```text
Ввід:
600
python15

Вивід:
Знижка: 90.0
До сплати: 510.0
```

Рішення:

```python
total = float(input("Сума покупки: "))
promo = input("Промокод: ").strip().lower()

has_discount = (total >= 500 and promo == "python15") or total >= 2000

if has_discount:
    discount = total * 0.15
else:
    discount = 0

final_total = total - discount

print("Знижка:", discount)
print("До сплати:", final_total)
```

## Вправа 18. Розумна перевірка замовлення

Інструкція: користувач вводить назву товару, кількість і ціну. Замовлення коректне, якщо назва не порожня, кількість більша за 0, ціна більша за 0. Якщо замовлення коректне, програма рахує суму. Інакше виводить повідомлення про помилку.

Приклад вводу/виводу:

```text
Ввід:
Кава
3
45.5

Вивід:
Замовлення прийнято
Товар: Кава
До сплати: 136.5
```

Рішення:

```python
product = input("Товар: ").strip()
quantity = int(input("Кількість: "))
price = float(input("Ціна: "))

if product and quantity > 0 and price > 0:
    total = quantity * price
    print("Замовлення прийнято")
    print("Товар:", product)
    print("До сплати:", total)
else:
    print("Помилка в замовленні")
```
