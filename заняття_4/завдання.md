## Вправа 12. Майстерня відладки логіки

Інструкція: наданий код містить помилки з логічними операторами, `=`, відступами, регістром змінних і порожнім рядком. Потрібно знайти й виправити всі помилки.

```python
name = input("Ім'я: ").strip()
age = int(input("Вік: "))
agree = input("Погоджуєшся? ").strip().lower()

if name and age >= 18 or agree = "так":
print("Реєстрацію дозволено")
else
    print("Реєстрацію заборонено")
print("Користувач:", Name)
```

Приклад вводу/виводу після виправлення:

```text

так

Вивід:

Користувач: Оля
```
