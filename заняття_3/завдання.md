## Вправа 1. Порівняння як питання

Інструкція: створити дві змінні з числами. Вивести результати порівнянь:

- перше число більше за друге;
- перше число менше за друге;
- числа рівні;
- числа не рівні;
- тип одного результату порівняння.

Приклад вводу/виводу:

```text

Вивід:
10 > 7: True
10 < 7: False
10 == 7: False
10 != 7: True
Тип результату: <class 'bool'>
```

## Вправа 6*. Розширена перевірка пароля

Інструкція: користувач вводить пароль. Програма має перевірити, чи пароль відповідає кільком умовам безпеки.

Пароль вважається надійним, якщо:

- його довжина не менша за 8 символів;
- він містить хоча б одну велику літеру;
- він містить хоча б одну малу літеру;
- він містить хоча б одну цифру;
- він не містить пробілів.

Програма має вивести результат перевірки. Якщо пароль не підходить, потрібно показати, які саме умови не виконані.

Приклад вводу/виводу:
```text
Ввід:
qwerty

Вивід:
Пароль ненадійний
Пароль закороткий
Немає великої літери
Немає цифри
Довжина пароля: 6
```
Інший приклад:
```text
Ввід:
Qwerty123

Вивід:
Пароль надійний

```
