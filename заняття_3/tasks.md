## Вправа 1. Порівняння як питання

Інструкція: створити дві змінні з числами. Вивести результати порівнянь:

- перше число більше за друге;
- перше число менше за друге;
- числа рівні;
- числа не рівні;
- тип одного результату порівняння.

Приклад вводу/виводу:

```text
Ввід: немає

Вивід:
10 > 7: True
10 < 7: False
10 == 7: False
10 != 7: True
Тип результату: <class 'bool'>
```

Рішення:

```python
first_number = 10
second_number = 7

print("10 > 7:", first_number > second_number)
print("10 < 7:", first_number < second_number)
print("10 == 7:", first_number == second_number)
print("10 != 7:", first_number != second_number)
print("Тип результату:", type(first_number > second_number))
```

## Вправа 6*. Розширена перевірка пароля

Інструкція: користувач вводить пароль. Програма має перевірити, чи пароль відповідає кільком умовам безпеки.

Пароль вважається надійним, якщо:

- його довжина не менша за 8 символів;
- він містить хоча б одну велику літеру;
- він містить хоча б одну малу літеру;
- він містить хоча б одну цифру;
- він не містить пробілів.

Програма має вивести результат перевірки. Якщо пароль не підходить, потрібно показати, які саме умови не виконані.

Приклад вводу/виводу:
```text
Ввід:
qwerty

Вивід:
Пароль ненадійний
Пароль закороткий
Немає великої літери
Немає цифри
Довжина пароля: 6
```
Інший приклад:
```text
Ввід:
Qwerty123

Вивід:
Пароль надійний
Довжина пароля: 9
```

Рішення:
```python
password = input("Пароль: ")

is_valid = True

if len(password) < 8:
  print("Пароль закороткий")
  is_valid = False

if not any(char.isupper() for char in password):
  print("Немає великої літери")
  is_valid = False

if not any(char.islower() for char in password):
  print("Немає малої літери")
  is_valid = False

if not any(char.isdigit() for char in password):
  print("Немає цифри")
  is_valid = False

if " " in password:
  print("Пароль не повинен містити пробілів")
  is_valid = False

if is_valid:
  print("Пароль надійний")
else:
  print("Пароль ненадійний")

print("Довжина пароля:", len(password))
```

Інструкція: користувач вводить суму покупки. Якщо сума 1000 або більше, програма дає знижку 10%. Інакше знижки немає. Вивести суму знижки і суму до сплати.

Ввід:

## Вправа 10. Калькулятор з вибором операції

Інструкція: користувач вводить два числа і операцію: `+`, `-`, `*` або `/`. Програма виконує потрібну дію. Якщо операція невідома, вивести `Невідома операція`.

Приклад вводу/виводу:

```text
Ввід:
8
3
*

Вивід:
Результат: 24.0
```

Рішення:

```python
first_number = float(input("Перше число: "))
second_number = float(input("Друге число: "))
operation = input("Операція (+, -, *, /): ").strip()

if operation == "+":
    result = first_number + second_number
    print("Результат:", result)
elif operation == "-":
    result = first_number - second_number
    print("Результат:", result)
elif operation == "*":
    result = first_number * second_number
    print("Результат:", result)
elif operation == "/":
    result = first_number / second_number
    print("Результат:", result)
else:
    print("Невідома операція")
```

## Вправа 18. Мінібанкомат

Інструкція: користувач вводить баланс і суму зняття. Якщо сума зняття менша або дорівнює балансу, програма дозволяє зняття і показує новий баланс. Інакше виводить `Недостатньо коштів`.

Приклад вводу/виводу:

```text
Ввід:
1000
350

Вивід:
Операцію дозволено
Новий баланс: 650.0
```

Рішення:

```python
balance = float(input("Баланс: "))
withdraw = float(input("Сума зняття: "))

if withdraw <= balance:
    balance = balance - withdraw
    print("Операцію дозволено")
    print("Новий баланс:", balance)
else:
    print("Недостатньо коштів")
```
