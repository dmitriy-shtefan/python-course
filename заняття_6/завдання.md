# Підготовчі завдання: навіщо потрібні списки

Ці завдання можна виконати без списків через окремі змінні. Але зі списками рішення стає коротшим, зрозумілішим і легше розширюється.

## Підготовче завдання 1. Студенти групи

Інструкція: задати студентів групи вручну в коді, без `input`, і вивести їх. Без списку це можна зробити через окремі змінні; зі списком усі студенти зберігаються в одній змінній.

Приклад вводу/виводу:

```text
Ввід: немає

Вивід:
Студенти: ['Оля', 'Максим', 'Ірина']
```

## Підготовче завдання 2. Перевірка дозволеної команди

Інструкція: користувач вводить команду. Дозволені команди: `start`, `help`, `exit`. Програма має перевірити, чи команда дозволена. Без списку доведеться писати кілька порівнянь через `or`; зі списком можна використати `in`.

Приклад вводу/виводу:

```text
Ввід:
help

Вивід:
Команда дозволена
```

## Підготовче завдання 3. Вибір студента за номером

Інструкція: є три студенти: Оля, Максим, Ірина. Користувач вводить номер від 1 до 3. Програма має вивести ім'я студента. Без списку це можна зробити через `if/elif`; зі списком достатньо перетворити номер на індекс.

Приклад вводу/виводу:

```text
Ввід:
2

Вивід:
Студент: Максим
```

## Підготовче завдання 4. Заміна статусу задачі

Інструкція: є три статуси задач: `нове`, `в роботі`, `нове`. Потрібно змінити статус другої задачі на `готово` і вивести всі статуси. Без списку потрібно змінювати окрему змінну; зі списком можна змінити елемент за індексом.

Приклад вводу/виводу:

```text
Ввід: немає

Вивід:
['нове', 'готово', 'нове']
```

## Підготовче завдання 5. Видалення останнього повідомлення

Інструкція: є три повідомлення. Потрібно видалити останнє і показати ті, що залишились. Без списку доводиться просто не виводити одну зі змінних; зі списком можна реально прибрати елемент через `pop`.

Приклад вводу/виводу:

```text
Ввід: немає

Вивід:
Видалено: третє повідомлення
['перше повідомлення', 'друге повідомлення']
```

## Вправа 18. Збери програму зі списком

Інструкція: рядки коду надані в неправильному порядку. Потрібно скласти програму, яка створює список, додає товар і виводить кількість товарів.

Перемішані рядки:

```python
print("Кількість:", len(products))
products.append(product)
products = ["хліб", "молоко"]
product = input("Новий товар: ").strip()
```

Приклад вводу/виводу:

```text
Ввід:
кава

Вивід:
Кількість: 3
```

## Вправа 19. Майстерня відладки списку

Зламаний код:

```python
products = ["хліб", "молоко"]
product = input("Товар: ").strip().lower()

products.add(product)

if "кава" in product:

else:
    print("Кави немає")

print("Перший товар:", Products[1])

```
