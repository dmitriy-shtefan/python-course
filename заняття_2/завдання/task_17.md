**Вправа 17. Мініаналізатор речення**

Інструкція: користувач вводить речення. Програма має прибрати пробіли на краях, вивести речення у нижньому регістрі, у верхньому регістрі, перші 10 символів, останні 10 символів, довжину речення і версію речення, де всі пробіли замінені на `_`.

Приклад вводу/виводу:

```text
Ввід:
  Python працює з текстом

Вивід:
Очищено: Python працює з текстом
Нижній регістр: python працює з текстом
Верхній регістр: PYTHON ПРАЦЮЄ З ТЕКСТОМ
Перші 10 символів: Python пра
Останні 10 символів:  з текстом
Довжина: 23
Для URL: Python_працює_з_текстом
```
