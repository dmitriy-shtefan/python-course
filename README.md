# Python Course

Навчальний проєкт для вивчення основ програмування мовою Python.

Репозиторій містить матеріали занять, презентації та практичні завдання для студентів, які починають програмувати з нуля або повторюють базові теми.

## Структура проєкту

```text
.
├── заняття_1/
│   ├── презентація/
│   └── завдання/
├── заняття_2/
│   ├── презентація/
│   └── завдання/
├── заняття_3/
│   ├── презентація/
│   └── завдання.md
└── заняття_4/
    └── завдання.md
...
```

## Теми занять

| Заняття | Тема | Практика |
| --- | --- | --- |
| 1 | Перший запуск Python, `print()`, `input()`, прості помилки | 10 вправ |
| 2 | Змінні, типи даних, перетворення типів, рядки, базові операції | 18 вправ |
| 3 | Порівняння, `bool`, `if`, `elif`, `else`, логіка вибору | 11 вправ |
| 4 | Логічні оператори, складні умови, `bool()`, проста валідація введення | 12 вправ |

## Як працювати з матеріалами

1. Відкрийте папку потрібного заняття.
2. Перегляньте презентацію, якщо вона є.
3. Виконуйте завдання по черзі.
4. Для кожної вправи створюйте окремий Python-файл, наприклад `task_1.py`.
5. Запускайте програму та порівнюйте результат із прикладом вводу/виводу в умові.

## Рекомендоване середовище

- Python 3.11 або новіший
- VS Code, PyCharm або інший редактор коду
- Термінал для запуску програм

Перевірити встановлену версію Python:

```bash
python3 --version
```

Запустити файл із програмою:

```bash
python3 task_1.py
```

## Для студентів

Під час виконання завдань важливо не тільки отримати правильний результат, а й розуміти:

- які дані вводить користувач;
- який тип має кожне значення;
- де потрібне перетворення типів через `int()`, `float()` або `str()`;
- як працюють умови та логічні оператори;
- чому виникають помилки та як їх виправляти.

Якщо програма не запускається, уважно перевірте:

- лапки та дужки;
- назви змінних;
- відступи в умовах;
- перетворення введених значень;
- повідомлення про помилку в терміналі.
